# ScaleFlow Skills

AI-powered creative production skills for Cowork by Claude. Built for creative agencies and production teams running campaigns with AI tools.

## 3 Plugin Packs — 11 Skills

### Creative Suite
| Skill | What It Does |
|-------|-------------|
| Brief Analyzer | Breaks down client briefs into structured creative direction |
| Copy Engine | Generates platform-specific ad copy with brand voice control |
| Moodboard Curator | Curates visual direction boards with reference imagery |
| Prompt Architect | Engineers prompts for Midjourney, DALL-E, Runway, Sora, etc. |
| Storyboard Writer | Creates shot-by-shot video storyboards with AI generation notes |

### Production Ops
| Skill | What It Does |
|-------|-------------|
| Credit Optimizer | Calculates AI platform credit budgets across tools |
| Creative Review | QA checklist for AI-generated assets before delivery |
| Asset Spec | Generates export specs for every platform and format |
| SOP Writer | Documents repeatable workflows as standard operating procedures |

### Client Delivery
| Skill | What It Does |
|-------|-------------|
| Deck Creator | Builds client-ready presentation decks |
| Report Builder | Generates post-project reports with usage analytics |

## Install in Cowork

1. Open **Cowork** → **Settings** → **Plugins**
2. Add marketplace: `scaleflow-solutions/scaleflow-skills`
3. Install the plugin packs you need
